Reproduire le vaisseau spatial

Image d'un vaisseau spatial généré avec par l'export d'un fichier HTML depuis le logiciel Inkscape, image de départ étant un pictogramme SVG. Dans le script sont générés les caractéristiques géométriques du dessin comme les coordonnées, les courbes de béziers, les chemins et les couleurs.
